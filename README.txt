Export Selected To Obj 1.0
==========================
modo Python script by Johan Steen.


Description
===========
Exports the selected layer(s) to an obj file.


Usage
=====
For simplicity you can assign the script to a button for easy access to it.


Parameters
==========
* None


Installation
============
* Copy the js_exportSelectedToObj.py to your modo user scripts folder.
